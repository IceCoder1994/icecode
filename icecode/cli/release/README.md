# icecode

**IceCoder's customized Freebuff.** A free AI coding agent that runs in your terminal — describe what you want, and it edits your code.

## Install

```bash
npm install -g icecode
```

## Usage

```bash
cd ~/my-project
icecode
```

## Download Binary

Pre-built binaries are available from the [GitHub Releases](https://github.com/IceCoder1994/icecode/releases) page.
